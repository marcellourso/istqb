# Monorepo

Struttura:
- `backend/`: FastAPI
- `frontend/`: React

## Prerequisiti
- Python 3.11+
- Node.js 20+
- (opzionale) `make`

## Avvio

### Backend
Da root repo:
- `make run-backend`

### Frontend
Da root repo:
- `make run-frontend`

## Test (placeholder)
- `make test`
- `make lint`
- `make e2e`
