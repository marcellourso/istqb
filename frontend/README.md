# Frontend (React)

## Prerequisiti

- Node.js 22+ (o Node >= 20.19)
- npm

## Config

Copia `.env.example` in `.env` e imposta il backend:

```bash
cp .env.example .env
```

Variabili:
- `VITE_API_BASE_URL` (es. `http://127.0.0.1:8000`)

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
